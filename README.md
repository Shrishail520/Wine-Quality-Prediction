# Wine Quality Prediction 🍷

## Project Overview
This project predicts whether a wine is of **Good** or **Bad** quality using Machine Learning techniques.

## Dataset
- Wine Quality Dataset
- Features: Acidity, Sugar, pH, Alcohol, Sulphates, etc.
- Target:
  - Good Wine (Quality ≥ 7)
  - Bad Wine (Quality < 7)

## Machine Learning Models
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree

## Project Workflow
1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Train-Test Split
5. Logistic Regression (Before Scaling)
6. StandardScaler
7. Logistic Regression (After Scaling)
8. Model Comparison
9. Hyperparameter Tuning using GridSearchCV
10. Feature Importance Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Results
The best-performing model was selected after comparing Logistic Regression, KNN, and Decision Tree and applying hyperparameter tuning.
